# Noir Archive - 题目15：博客前台 + 后台系统

本项目是基于 Vue 3 + Vite 实现的博客前台与后台管理系统，完成了文章浏览、文章详情、分类筛选、搜索、后台登录、文章发布、文章编辑、文章删除、分类管理、Markdown 编辑器等功能。

## 技术栈

- Vue 3
- Vite
- Vue Router
- Pinia
- Axios
- md-editor-v3
- VueUse
- Vitest
- Vercel

## 功能说明

### 前台功能

- 首页精选文章展示
- 文章列表
- 文章详情
- 分类筛选
- 文章搜索
- 分类集合页

### 后台功能

- 登录认证
- 路由守卫
- 文章发布
- 文章编辑
- 文章删除
- 分类新增
- 分类重命名
- 分类删除
- Markdown 编辑器实时预览
